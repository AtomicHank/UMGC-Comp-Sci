2. Floyd's algorithm on the graph from Problem 1

Use edge length 1 for each edge, 0 on the diagonal, and infinity where no edge exists.

Initial distance matrix:

[0   1  inf inf inf
 inf 0  inf 1   inf
 inf inf 0   1   inf
 inf inf inf 0   1
 inf inf 1   inf 0]

Final Floyd result:

[0   1   4   2   3
 inf 0   3   1   2
 inf inf 0   1   2
 inf inf 2   0   1
 inf inf 1   2   0]

Interpretation:

- From a, the shortest path reaches every other vertex.
- From b, the reachable vertices are d, e, and c.
- Vertices c, d, and e can all reach one another.


3. Graph, condensation graph, topological order, and closure

Graph corresponding to the matrix in Problem 1:

Vertices:
a, b, c, d, e

Edges:
a -> b
b -> d
c -> c
c -> d
d -> e
e -> c
e -> e

Strongly connected components:

{a}
{b}
{c, d, e}

Condensation graph:

Let

X = {a}
Y = {b}
Z = {c, d, e}

Then the condensation graph has edges:

X -> Y
Y -> Z

One topological order is:

X, Y, Z

Adjacency matrix in that topological order:

[0 1 0
 0 0 1
 0 0 0]

Reflexive-transitive closure of that matrix:

[1 1 1
 0 1 1
 0 0 1]

Characteristic indicating a total order:

Every entry on and above the main diagonal is 1. That means for any two vertices in this topological order, one is always related to the other, so every pair is comparable. That is the defining property of a total order.
